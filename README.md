# Акинатор для дерматовенеролога

Сделана расширенная версия с большой базой знаний:

- много диагнозов (дерматология + ИППП),
- ICD-10 коды,
- профильные тесты и первичная тактика,
- объяснение, почему гипотеза попала в топ.

## Запуск

```bash
python3 akinator_derm.py
```

## Структура проекта

- `akinator_derm.py` - движок вопросов и ранжирования.
- `knowledge_base.py` - большая база знаний (состояния, профили, теги симптомов, рекомендации).

## Как работает скоринг

1. Пользователь проходит опрос (локализация, морфология, зуд, боль, ИППП-факторы риска и т.д.).
2. Ответы формируют набор симптом-тегов с весами.
3. Для каждого состояния считаются совпадения тегов с базой знаний.
4. Применяются condition-specific поправки (бонусы/штрафы) для похожих диагнозов.
5. Итоговый score корректируется коэффициентом покрытия признаков (precision-friendly).
6. После базового шага включается адаптивный этап (1-3 уточняющих вопроса для текущих лидеров).
7. Рассчитывается уровень уверенности (`низкая`/`средняя`/`высокая`) и объяснение выбора.
8. Выводится наиболее вероятный вариант (топ-1) + что проверить и базовая тактика.

Дополнительно для точности:
- при низкой информативности ответов (много `не знаю`/`неприменимо`) гипотеза не форсируется;
- исключаются кандидаты с недостаточным числом совпавших признаков;
- уточняющие вопросы задаются только когда они релевантны текущему профилю симптомов.
- часть базовых вопросов теперь тоже задается по ветвлению (только если клинически релевантны).

## Источники, на которые опирается база

- WHO ICD-10: блоки `L00-L99` (кожа и подкожная клетчатка).
- WHO ICD-10: блок `A50-A64` (венерические болезни).
- CDC STI Treatment Guidelines 2021.
- DermNet диагностический индекс.

## Как расширять

1. Добавить новое состояние в `RAW_CONDITIONS` (`knowledge_base.py`).
2. При необходимости создать/доработать профиль в `PROFILES`.
3. Добавить новый тег в вопросы (`QUESTIONS`) и в `humanize_tag` в `akinator_derm.py`.

## Важно

Инструмент учебно-скрининговый и не заменяет очный осмотр, лабораторную диагностику и клиническое решение врача.
